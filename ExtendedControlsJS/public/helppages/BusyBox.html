Busy Box

Description

Provides an animated busy box

HTML Attibutes

Javascript arguments

text:
The text to display at the bottom of the box
title:
The text to display at the top of the box
show:
Show the busy box on instantiation
align:
Align the box in the parent (centre, left, right)
boxstyle:
The style of the busy box (default, pleasewait)
inverse:
Invert the colours of the box
spinner:
The CSS class of the icon to use as the spinner
